# Data Science & Machine Learning Toolkit

This repository contains a comprehensive assignment covering the essential libraries for Data Science in Python.

## 🛠️ Modules Covered
- **NumPy:** Multi-dimensional array operations, broadcasting, and linear algebra.
- **Pandas:** Data cleaning, transformation, and statistical exploration of structured data.
- **Scikit-Learn:** Binary classification and performance metrics (Confusion Matrix).

## 🚀 Key Tasks
1. **Array Engineering:** Handling 2D arrays, reshaping matrices, and performing vectorized operations.
2. **HR Data Analysis:** Creating a mock employee database, calculating bonuses, and departmental filtering.
3. **Iris Classification:** Implementing a Logistic Regression model to classify 'Versicolor' species and evaluating results using a Confusion Matrix.

## 📊 Evaluation Metrics Explained
- **Accuracy:** Overall correctness of the model.
- **Precision:** Accuracy of positive predictions.
- **Recall:** Ability to find all positive instances.
- **F1-Score:** Harmonic mean of Precision and Recall.
